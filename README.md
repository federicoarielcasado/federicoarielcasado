#### 📌 Featured Projects

🧮 **[PyStrAn](https://github.com/federicoarielcasado/py-str-an)** — Structural Analysis Software  
Desktop application for 2D hyperstatic structural analysis using the Deformation Method and the
Force Method, with cross-validated results. Drag-and-drop interface, stress diagrams (M, V, N)
and elastic deformed shape. 341 automated tests.  
`Python` `PyQt6` `NumPy` `SciPy` `Matplotlib`

🛰️ **[py-sat-meas](https://github.com/federicoarielcasado/py-sat-meas)** — Satellite Rooftop Detection  
Automated detection and measurement of roofed surfaces from Sentinel-2 L2A satellite imagery.
Dual interchangeable engine: spectral indices (NDVI/NDBI/NDWI) + U-Net neural network (PyTorch)
with local fine-tuning. Copernicus Data Space (CDSE) API integration.  
`Python` `PyTorch` `GDAL` `GeoPandas` `OpenCV`

📐 **[py-param-cad](https://github.com/federicoarielcasado/py-param-cad)** — Parametric CAD Design  
Automated generation of editable 3D models (.FCStd/.STEP) and 2D manufacturing drawings (DXF/PDF)
from technical parameters. 11 real-time AISC/IRAM validation rules, IRAM 4505 title blocks,
exportable bill of materials. 151 automated tests. v1.0.0 released.  
`Python` `PyQt6` `FreeCAD` `ezdxf` `SQLAlchemy`
